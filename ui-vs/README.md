Giao diện mẫu — Welcome page (đen & trắng)

Mô tả nhanh:
- Đây là một mẫu trang welcome tối giản cho hệ thống quản lý thư viện.
- Tập trung vào layout đen-trắng, typography rõ ràng, và CTA (Bắt đầu ngay / Tìm hiểu).

Cách xem:
1. Mở file `index.html` trong trình duyệt (double-click hoặc `open index.html`).
2. Hoặc từ terminal (Linux):

```bash
xdg-open index.html
```

Muốn thay đổi? Tôi có thể:
- Thêm phiên bản dark-mode/contrast cao hơn.
- Chuyển sang grid sách động hoặc thêm hình minh họa SVG chi tiết.
- Tạo component React/Vue nếu cần tích hợp vào app.
